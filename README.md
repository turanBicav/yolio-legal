# Yolio Legal Site

Yolio uygulamasının gizlilik politikası, destek sayfası ve genel marka bilgisi.
GitHub Pages üzerinde host edilir.

## Dosyalar

- `index.md` — Ana sayfa
- `privacy.md` — Gizlilik politikası (TR + EN)
- `support.md` — Destek sayfası (TR + EN)
- `_config.yml` — Jekyll konfigürasyonu

## Yayın URL'leri

Aktif olduktan sonra:

- Ana: `https://<kullanıcı-adı>.github.io/yolio-legal/`
- Privacy: `https://<kullanıcı-adı>.github.io/yolio-legal/privacy/`
- Support: `https://<kullanıcı-adı>.github.io/yolio-legal/support/`

(Eğer custom domain bağlanırsa: `https://yolio.app/privacy/` gibi.)

## Güncelleme

Dosyaları düzenle → commit → push → ~1 dakikada canlıya çıkar.
